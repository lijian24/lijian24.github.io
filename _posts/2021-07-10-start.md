## 配置环境记录
```shell
1. git clone from your repository
2. mac下环境配置：
    brew install ruby
    gem install jekyll bundler
    bundle config set --local path 'vendor/bundle'
    bundle install
3. 服务启动
    bundle exec jekyll serve

注意：更换模板之后，需要删除vendor/bundle，并重新执行bundle install，用来下载所有依赖的环境包。
```
